# 🐹 Desktop Pet — 나만의 데스크탑 펫 만들기

당신의 Mac 화면 위에서 귀엽게 뛰어다니는 캐릭터! 🎉  
**Desktop Pet**은 Python과 PyObjC를 이용해 만든 맞춤형 데스크탑 펫 앱입니다.

## ✨ 주요 기능

- 💾 **투명한 배경의 PNG 이미지**로 자신만의 캐릭터 만들기
- 🖱️ **마우스 따라다니기** 모드 — 펫이 당신의 마우스를 쫓아다닙니다 (선택 사항)
- 🚶 **자유 배회** 모드 — 펫이 화면을 돌아다니면서 휴식을 취합니다
- 🎬 **부드러운 애니메이션** — 두 장의 프레임으로 뛰는 모션 표현
- 📍 **모든 화면 공간 사용** — Spaces와 Mission Control에서도 계속 활동
- 🎮 **GUI 런처** — 직관적인 인터페이스로 쉽게 시작하기

## 🚀 시작하기

### 필수 요구사항

- Python 3.7 이상
- Mac OS (PyObjC 기반)
- PIL/Pillow (이미지 처리용)

### 설치

```bash
# 1. 저장소 클론
git clone <repository-url>
cd <project-directory>

# 2. 필요한 패키지 설치
pip install -r requirements.txt
```

### 펫 실행하기

```bash
# GUI 런처로 실행
python launcher.py
```

런처 창에서:
1. **프레임 1** — 첫 번째 뛰는 모션 이미지 선택 (PNG)
2. **프레임 2** — 두 번째 뛰는 모션 이미지 선택 (PNG)
3. **마우스 따라다니기** — 활성화/비활성화 선택
4. **펫 실행하기** 버튼 클릭 🎊

## 🎨 이미지 가이드

펫을 만들기 위해 다음 사항을 지켜주세요:

| 요구사항 | 설명 |
|---------|------|
| **파일 형식** | PNG (투명한 배경 필수) |
| **프레임** | 연속된 두 장의 뛰는 동작 이미지 필요 |
| **배경** | **반드시 투명해야 합니다** (알파 채널 필수) |
| **크기** | 너무 크지 않은 적당한 크기 권장 (예: 150x150px ~ 300x300px) |
| **형태** | 왼쪽 → 오른쪽 동작이 자연스러워야 함 |

### 예시

```
프레임 1 (첫 번째 동작)  →  프레임 2 (두 번째 동작)
     🐹                         🐹
  (왼발 앞)                  (오른발 앞)
```

> 💡 **팁**: 애니메이션 소프트웨어(Aseprite, Piskel 등)를 사용해 만드는 것을 추천합니다!

## 🔨 자체 빌드하기 (Executable 만들기)

`pyinstaller`를 사용해 독립 실행형 앱으로 빌드할 수 있습니다:

```bash
# 1. pyinstaller 설치
pip install pyinstaller

# 2. 빌드 실행
pyinstaller --onefile --windowed --icon=icon.icns launcher.py

# 3. 완성! 
# dist/launcher 가 생성되며, 이를 더블클릭하면 런처가 실행됩니다 ✨
```

생성된 실행 파일은 `dist/` 폴더에서 찾을 수 있습니다.

> 📦 **참고**: 프로젝트의 `icon.icns` 파일을 아이콘으로 사용합니다. 자신의 아이콘으로 바꾸고 싶다면 `--icon=your-icon.icns`로 변경하세요.

## ⚙️ 사용자 정의

코드 상단의 설정값을 수정해 펫의 동작을 조정할 수 있습니다:

```python
# desktop_pet.py 내 설정값
PET_SCALE = 0.08              # 펫 크기 (0.01 ~ 1.0)
MOVE_SPEED = 4                # 이동 속도
ANIM_INTERVAL = 0.3           # 애니메이션 프레임 전환 속도 (초)
IDLE_CHANCE = 0.003           # 휴식할 확률
IDLE_DURATION = 2.0           # 휴식 시간 (초)
FOLLOW_SPEED = 3              # 마우스 추적 속도
FOLLOW_THRESHOLD = 30         # 마우스까지의 거리 (픽셀)
```

## 🎮 작동 방식

### 마우스 따라다니기 모드 (`follow_mouse=True`)
- 펫이 마우스 커서를 추적합니다
- 일정 거리 내에 도착하면 멈추고 기다립니다
- 펫이 항상 마우스 방향을 바라봅니다

### 자유 배회 모드 (`follow_mouse=False`)
- 펫이 화면을 이리저리 돌아다닙니다
- 간헐적으로 멈춰서 휴식을 취합니다
- 화면 경계에 닿으면 자동으로 반대 방향으로 이동합니다
